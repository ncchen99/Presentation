# Wireframe — Dashboard 頁面

## 頁面用途
使用者登入後的主要工作台，一眼看見最重要的狀態概覽，並快速進入各子功能。

## 版面配置

```
┌─────────────────────────────────────────────────────┐
│  [Logo]   [Nav: 首頁 / 歷史 / 設定]      [用戶頭像] │  ← Header（固定頂部）
├──────────┬──────────────────────────────────────────┤
│          │  歡迎回來，{用戶名稱}                    │
│  Sidebar │  ─────────────────────────────────────── │
│          │  [統計卡片 1]  [統計卡片 2]  [統計卡片 3]│  ← 統計摘要列
│  [快捷   │                                          │
│   功能   │  ─── 最近活動 ────────────────────────── │
│   清單]  │  [活動項目 1]                            │
│          │  [活動項目 2]                            │
│          │  [活動項目 3]                            │
│          │  [載入更多] 按鈕                         │  ← 主要內容區
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

## 各區塊說明

### Header
- Logo：點擊回首頁
- 主導航列：目前所在頁面需有 active 樣式
- 用戶頭像：點擊展開下拉選單（個人資料 / 登出）

### Sidebar
- 快捷功能清單：根據用戶角色顯示不同項目
- 目前頁面以高亮色標示

### 統計摘要卡片（3 欄）
- 每張卡片顯示：數字 + 標籤 + 趨勢箭頭（↑/↓）
- 點擊可進入對應的詳細頁面

### 最近活動列表
- 每筆活動：時間戳 + 操作描述 + 狀態標籤
- 預設顯示最新 10 筆，超過可點「載入更多」
- 空白狀態：顯示「還沒有任何活動，開始第一個吧！」圖示 + 文字

## 互動元件

| 元件 | 行為 |
|------|------|
| 用戶頭像 | 點擊展開下拉選單 |
| 統計卡片 | 點擊導向詳細頁面 |
| 活動項目 | 點擊展開詳細資訊（in-place 或側邊抽屜） |
| 載入更多按鈕 | 追加載入下一批，按鈕消失若已全部顯示 |

## 狀態處理

- **載入中**：各卡片顯示 skeleton loading 動畫
- **空白狀態**：活動列表無資料時顯示引導圖示
- **錯誤狀態**：API 失敗時卡片顯示「載入失敗，點擊重試」
